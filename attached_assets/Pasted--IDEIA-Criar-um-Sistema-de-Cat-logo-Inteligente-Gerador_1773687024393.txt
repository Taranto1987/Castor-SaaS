## IDEIA

Criar um **Sistema de Catálogo Inteligente + Gerador de Orçamento para a Loja Castor Cabo Frio**.

O sistema faz três coisas automaticamente:

1. **Crawler do site Castor**

   * varre as categorias que você vende
   * coleta todos os produtos
   * salva os dados em um banco interno

2. **Banco de Produtos da Loja**

   * nome do produto
   * SKU / referência
   * preço normal
   * preço Pix
   * parcelamento
   * medidas
   * altura
   * link
   * imagem

3. **Gerador de Orçamento**

   * busca produto no banco
   * gera texto de orçamento
   * entrega pronto para copiar e enviar no WhatsApp

Resultado prático:

Você digita ou fala:

```
orçamento João Pedro
Amazon Gel King
```

O sistema retorna:

```
ORÇAMENTO – CASTOR CABO FRIO

Cliente: João Pedro

Produto:
Colchão Castor Amazon Premium Gel Double Face Pocket King

Preço: R$ 7.410,04
Pix: R$ 6.298,53
Parcelamento: 12x de R$ 617,50

WhatsApp: (22) 99241-0112
```

Sem abrir site.
Sem procurar preço.

# BLUEPRINT DO SISTEMA

## Objetivo

Criar uma aplicação Node.js que:

1. Extrai produtos da Castor
2. Cria banco de dados
3. Permite buscar produto
4. Gera orçamento automaticamente

# PROMPT PARA O REPLIT

Cole este prompt no **Replit AI Agent**:

```
Create a Node.js application that builds a product catalog for a mattress store using data from the website lojacastor.com.br.

The system must:

1. Crawl the website using Playwright.
2. Extract all products from the following categories:

https://lojacastor.com.br/colchoes
https://lojacastor.com.br/cama-box
https://lojacastor.com.br/cama-box-colch-o
https://lojacastor.com.br/travesseiros
https://lojacastor.com.br/roupa-de-cama
https://lojacastor.com.br/roupa-de-cama/protetor

Ignore these categories:

bem-estar
estofados
moveis
linha-pet

For each product extract:

product name
SKU
price
pix price
installment information
dimensions
height
category
image
product URL

Store all products in a SQLite database.

Also export the catalog to JSON and CSV.

Then build an Express API with two endpoints:

/search?q=product_name
Returns product information from the database.

/quote?cliente=name&produto=name
Returns a formatted sales quote text ready to send to a customer via WhatsApp.

Use Playwright for crawling and better-sqlite3 for the database.
```
