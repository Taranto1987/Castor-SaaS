# SUMÁRIO

Sistema atual travado por ausência de ordem de execução, inconsistência no núcleo financeiro e ausência de integração entre módulos.

Solução: **reconstrução controlada do CASTOR OS por pipeline sequencial obrigatório**, onde cada bloco é fechado, validado e somente então libera o próximo.

Resultado: **manual único, definitivo e executável**, contendo arquitetura completa, artefatos reais e ordem de implantação sem ambiguidade.

---

# SOLUÇÃO FINAL

Implantação do sistema completo dividido em **7 BLOCOS ESTRUTURAIS**, com dependência rígida:

```
[1] MOTOR DE ORÇAMENTO
[2] NORMALIZAÇÃO NUMÉRICA
[3] CONVERSÃO DE VENDA
[4] GERAÇÃO DE ENTREGA
[5] LOGÍSTICA E ROTEIRIZAÇÃO
[6] HUD + PERFORMANCE
[7] DADOS + CRAWLER + INTELIGÊNCIA
```

Cada bloco contém:

* arquitetura própria
* artefatos obrigatórios
* validação operacional
* travas de consistência

# BLOCO 1 — MOTOR DE ORÇAMENTO (CRÍTICO)

## REGRA CENTRAL

```
VALOR_BASE → DESCONTO (1x) → VALOR_FINAL → PARCELAMENTO
```

## IMPLEMENTAÇÃO

```javascript
// /engine/calc.js

export function calcularDesconto(valorBase, percentual) {
  return valorBase - (valorBase * (percentual / 100));
}

export function calcularParcela(valorFinal, parcelas) {
  return valorFinal / parcelas;
}
```

```javascript
// uso obrigatório

const valorBase = produto.preco_cheio;

const valorFinal = calcularDesconto(valorBase, desconto);

const valorParcela = calcularParcela(valorFinal, parcelas);

const valorPix = valorFinal;
```

## TRAVA

```javascript
let descontoAplicado = false;

export function aplicarDescontoSeguro(valor, percentual) {
  if (descontoAplicado) throw new Error("DESCONTO_DUPLICADO");
  descontoAplicado = true;
  return calcularDesconto(valor, percentual);
}
```

```javascript
export function formatarBRL(valor) {
  return new Intl.NumberFormat('pt-BR', {
    style: 'currency',
    currency: 'BRL'
  }).format(valor);
}

export function parseValorBR(valor) {
  return Number(valor.replace(/\./g, '').replace(',', '.'));
}
```

# BLOCO 3 — CONVERSÃO DE VENDA

```javascript
// /api/fechar-venda.js

import { v4 as uuid } from 'uuid';

export default async function handler(req, res) {
  const { orcamento_id } = req.body;

  const orcamento = await db('orcamentos')
    .where({ id: orcamento_id })
    .first();

  const venda_id = uuid();

  await db('vendas').insert({
    id: venda_id,
    orcamento_id,
    vendedor_id: orcamento.vendedor_id,
    valor: orcamento.valor
  });

  await db('orcamentos')
    .update({ status: 'VENDIDO' })
    .where({ id: orcamento_id });

  await criarEntrega(venda_id, orcamento);

  res.json({ ok: true });
}
```

# BLOCO 5 — LOGÍSTICA E ROTA

```javascript
// /engine/logistics.js

export async function gerarRota(origem, entregas) {
  const destinos = entregas.map(e => `${e.latitude},${e.longitude}`);

  const response = await axios.get(
    'https://maps.googleapis.com/maps/api/directions/json',
    {
      params: {
        origin: origem,
        destination: destinos[destinos.length - 1],
        waypoints: `optimize:true|${destinos.join('|')}`,
        key: process.env.GMAPS_KEY
      }
    }
  );

  return response.data.routes[0];
}
```

```javascript
export function calcularScore(m) {
  return (
    (m.conversao * 0.3) +
    ((m.faturamento / m.meta) * 0.25) +
    ((1 / m.tempoResposta) * 0.15) +
    ((m.followups / m.orcamentos) * 0.10) +
    ((m.ticket / m.mediaEquipe) * 0.10)
```
