# SUMÁRIO

Problema identificado no sistema atual do **Gerador de Orçamento Castor Cabo Frio**:

1. O sistema **não usa a mensagem padrão VIP da loja**.
2. O gerador permite **apenas um produto**, impossibilitando montar combos (ex.: cama baú + colchão + outro item).
3. A estrutura do orçamento **não segue o padrão de venda da loja**.

Solução implementada:

**Motor de orçamento multi-produto + template VIP fixo da Loja Castor Cabo Frio.**

---

# SOLUÇÃO FINAL

Substituir o gerador atual por:

### **Motor de Orçamento VIP Multi-Produto**

Capacidades:

```
• Selecionar múltiplos produtos
• Somar valores automaticamente
• Aplicar desconto PIX automático
• Gerar mensagem padrão VIP
• Copiar texto pronto para WhatsApp
```

Resultado:

```
1 cliente
+ vários produtos
= 1 orçamento formatado VIP
```

# MOTOR DE CÁLCULO

```ts
function calcularOrcamento(produtos){

 let totalPix = 0
 let totalPrazo = 0

 produtos.forEach(p=>{
   totalPix += p.preco_pix
   totalPrazo += p.preco_prazo
 })

 const parcela = totalPrazo / 12

 return {
   totalPix,
   totalPrazo,
   parcela
 }

}
```

# TEMPLATE PADRÃO VIP CASTOR CABO FRIO

Este template fica **fixo no sistema**.

 const calc = calcularOrcamento(produtos)

 const lista = produtos.map((p,i)=>{

 return `

Prazo: R$ ${p.preco_prazo}

 }).join("\n")

 return `

🇧🇷 CASTOR CABO FRIO

Cliente: ${cliente}

Produtos Selecionados:
${lista}

━━━━━━━━━━━━━━━━━━━━━━

Valor Total:

💰 PIX
R$ ${calc.totalPix.toFixed(2)}

💳 Parcelado
R$ ${calc.totalPrazo.toFixed(2)}

ou

━━━━━━━━━━━━━━━━━━━━━━

Observações:
${obs}

📞 WhatsApp Loja
(22) 99241-0112

}
